# How to set image index, time interval, categories

- ?v=0 : 0-index image
- ?v=100 : 100-index image
- ?v=r : random image, (default) refresh 20 seconds
- ?v=r&t=10000 : random image, refresh 10 seconds
- ?v=r&t=50000 : random image, refresh 50 seconds
- ?l=i : 'indoor'
- ?l=isung : 'indoor', 'studio', 'urban', 'nature', 'night' (any order / single alphabet)
